#CinePro
CinePro is a fully public, no-login-required web platform where users can discover movies across all genres and get torrent links instantly. Browse Hollywood blockbusters, Bollywood hits, anime, animations, and more — all in one place.



#Features

-  **Search & Filter Movies** — Find movies by title, genre, or rating in seconds
-  **User Ratings & Reviews** — See what the community thinks before you download
-  **All Genres Supported** — Action, Drama, Comedy, Horror,, Animation & more
-  **No Login Required** — Fully public, open access for everyone
-  **Lightning Fast** — Built with React + SWC for blazing performance

# Installation & Setup

## Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/cinepro-i.git
cd cinepro-i
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### 4. Build for Production

```bash
npm run build
```

### 5. Preview Production Build

```bash
npm run preview
```
